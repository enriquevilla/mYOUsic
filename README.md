# mYOUsic

### To run this project locally:

#### Prerequisites

* MongoDB
    * Default connection port is 27017*
    * Default DB name is mYOUsicDB*
* Node.js

#### Build commands

1. `npm install`
2. `npm run build`
3. `npm start`
4. App is running on port 8080*
